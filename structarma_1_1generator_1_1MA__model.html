Solve nonlinear system with fixed-point iteration algorithm to find moving-average coefficients \(\theta\).

Parameters
max_iterationsMaximal no. of iterations.
epsMaximal difference between values of white noise variance in successive iterations.
Parameters
min_var_wnMaximal value of white noise variance considered to be nought.
  1. Precompute white noise variance for the first iteration.
  2. Update coefficients from back to front using the following formula (adapted from G. Box and G. Jenkins (1970) "Time Series Analysis: Forecasting and Control", pp. 226–227).

    \[ \theta_{i,j,k} = -\frac{\gamma_0}{\sigma_\alpha^2} + \sum\limits_{l=i}^{n_1} \sum\limits_{m=j}^{n_2} \sum\limits_{n=k}^{n_3} \theta_{l,m,n} \theta_{l-i,m-j,n-k} \]

    Here \(\theta_{0,0,0} \equiv 0\).
  3. Zero out \(\theta_0\).
  4. Validate coefficients.
  5. Compute white noise variance by calling MA_model::white_noise_variance.
  6. Validate white noise variance.
  7. Print solver state.

  1. Precompute white noise variance for the first iteration.
  2. Update coefficients using the following formula (adapted from G. Box and G. Jenkins (1970) "Time Series Analysis: Forecasting and Control", p. 227).

    \[ \theta_{i,j,k} = -\frac{\tau_{i,j,k}}{\tau{0,0,0}}, f_{i,j,k} = \sum\limits_{l=0}^{n_1-i} \sum\limits_{m=0}^{n_2-j} \sum\limits_{n=0}^{n_3-k} \tau_{l,m,n} \tau_{l+i,m+j,n+k} - r_{i,j,k}. \]

    Here \(\tau_{0,0,0}^2 = \sigma_\alpha^2\).
  3. Zero out \(\theta_0\).
  4. Validate coefficients.
  5. Compute white noise variance by calling MA_model::white_noise_variance.
  6. Validate white noise variance.
  7. Print solver state.
