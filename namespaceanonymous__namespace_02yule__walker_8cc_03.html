Computes a part of Yule—Walker matrix \(R_{a,b} = \begin{bmatrix} \boldsymbol{\rho}_{a,a,a;b}\\ R_{a,b;1}\\ \vdots\\ R_{a,b;a}\\ \end{bmatrix}\).

Compute the first row, then for \(e=1,\ldots,a\) compute submatrices \(R_{a,b;e} = \begin{bmatrix} R_{a,b;e}^{(1)}\\ \vdots\\ R_{a,b;e}^{(6)}\\ \end{bmatrix}\) where

\(R_{a,b;e}^{(1)} = \begin{bmatrix} \boldsymbol{\rho}_{a-e,a,a;b}\\ \vdots\\ \boldsymbol{\rho}_{a-e,a-e+1,a;b}\\ \end{bmatrix}\),

\(R_{a,b;e}^{(2)} = \begin{bmatrix} \boldsymbol{\rho}_{a-e,a-e,a;b}\\ \vdots\\ \boldsymbol{\rho}_{a-1,a-e,a;b}\\ \end{bmatrix}\),

\(R_{a,b;e}^{(3)} = \begin{bmatrix} \boldsymbol{\rho}_{a,a-e,a;b}\\ \vdots\\ \boldsymbol{\rho}_{a,a-e,a-e+1;b}\\ \end{bmatrix}\),

\(R_{a,b;e}^{(4)} = \begin{bmatrix} \boldsymbol{\rho}_{a,a-e,a-e;b}\\ \vdots\\ \boldsymbol{\rho}_{a,a-1,a-e;b}\\ \end{bmatrix}\),

\(R_{a,b;e}^{(5)} = \begin{bmatrix} \boldsymbol{\rho}_{a,a,a-e;b}\\ \vdots\\ \boldsymbol{\rho}_{a-e+1,a,a-e;b}\\ \end{bmatrix}\),

\(R_{a,b;e}^{(6)} = \begin{bmatrix} \boldsymbol{\rho}_{a-e,a,a-e;b}\\ \vdots\\ \boldsymbol{\rho}_{a-e,a,a-1;b}\\ \end{bmatrix}\).

Computes a part of Yule—Walker matrix for \(b=0\).

Compute the first row, then for \(e=1,\ldots,a\) compute submatrices \(R_{a,0;e} = \begin{bmatrix} R_{a,0;e}^{(1)}\\ \vdots\\ R_{a,0;e}^{(6)}\\ \end{bmatrix}\) where

\(R_{a,0;e}^{(1)} = \begin{bmatrix} \rho_{a-e,a,a}\\ \vdots\\ \rho_{a-e,a-e+1,a}\\ \end{bmatrix}\),

\(R_{a,0;e}^{(2)} = \begin{bmatrix} \rho_{a-e,a-e,a}\\ \vdots\\ \rho_{a-1,a-e,a}\\ \end{bmatrix}\),

\(R_{a,0;e}^{(3)} = \begin{bmatrix} \rho_{a,a-e,a}\\ \vdots\\ \rho_{a,a-e,a-e+1}\\ \end{bmatrix}\),

\(R_{a,0;e}^{(4)} = \begin{bmatrix} \rho_{a,a-e,a-e}\\ \vdots\\ \rho_{a,a-1,a-e}\\ \end{bmatrix}\),

\(R_{a,0;e}^{(5)} = \begin{bmatrix} \rho_{a,a,a-e}\\ \vdots\\ \rho_{a-e+1,a,a-e}\\ \end{bmatrix}\),

\(R_{a,0;e}^{(6)} = \begin{bmatrix} \rho_{a-e,a,a-e}\\ \vdots\\ \rho_{a-e,a,a-1}\\ \end{bmatrix}\).

Computes row vector \(\boldsymbol{\rho}_{i,j,k;d} = \left( \rho_{i-d,j-d,k-d}, \boldsymbol{\rho}_{i,j,k;d,1}, \ldots, \boldsymbol{\rho}_{i,j,k;d,d} \right)\).

Compute the first element, then for \(e=1,\ldots,d\) compute sub-vectors \(\rho_{i,j,k;d,e} = \left( \boldsymbol{\rho}_{i,j,k;d,e}^{(1)}, \ldots, \boldsymbol{\rho}_{i,j,k;d,e}^{(6)} \right)\) where

\(\rho_{i,j,k;d,e}^{(1)} = \left( \rho_{i-d+e,j-d,k-d}, \ldots, \rho_{i-d+e,j-d+e-1,k-d} \right)\),

\(\rho_{i,j,k;d,e}^{(2)} = \left( \rho_{i-d+e,j-d+e,k-d}, \ldots, \rho_{i-d+1,j-d+e,k-d} \right)\),

\(\rho_{i,j,k;d,e}^{(3)} = \left( \rho_{i-d,j-d+e,k-d}, \ldots, \rho_{i-d,j-d+e,k-d+e-1} \right)\),

\(\rho_{i,j,k;d,e}^{(4)} = \left( \rho_{i-d,j-d+e,k-d+e}, \ldots, \rho_{i-d,j-d+1,k-d+e} \right)\),

\(\rho_{i,j,k;d,e}^{(5)} = \left( \rho_{i-d,j-d,k-d+e}, \ldots, \rho_{i-d+e-1,j-d,k-d+e} \right)\),

\(\rho_{i,j,k;d,e}^{(5)} = \left( \rho_{i-d+e,j-d,k-d+e}, \ldots, \rho_{i-d+e,j-d,k-d+1} \right)\).
