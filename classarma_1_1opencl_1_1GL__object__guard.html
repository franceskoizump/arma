Detailed Description

An object that manages shared OpenCL/OpenGL objects by keeping them acquired.

On construction it acquires shared OpenGL buffer by calling clEnqueueAcquireGLObjects, on destruction it releases the buffer by calling clEnqueueReleaseGLObjects. If OpenCL/OpenGL sharing is not supported, then no calls are made.
