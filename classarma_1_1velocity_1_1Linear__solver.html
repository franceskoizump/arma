[in]za coordinate \(z\) in which to compute velocity potential
[in]idx_ta time point in which to compute velocity potential, specified as index of zeta
  1. Compute window function.

    \[ \mathcal{W}(u, v) = 4\pi \frac{ \cosh\left(|\vec{k}|(z + h)\right) } { |\vec{k}|\cosh\left(|\vec{k}|h\right) } \]

  2. Compute \(\zeta_t\).
  3. Compute Fourier transforms.

    \[ \phi(x,y,z,t) = \text{Re}\left\{ \mathcal{F}_{x,y}^{-1}\left\{ \mathcal{W}(u, v) \mathcal{F}_{u,v}\left\{\zeta_t\right\} \right\} \right\} \]
